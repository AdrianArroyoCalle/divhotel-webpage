DivHotel

DivHotel is a simulation game about a hotel. With DivHotel you can control your hotel and manage it.

DivHotel is still on Alpha. At the moment we can't offer any way of pay but the price is (with updates): 20$

DivHotel is avalible in the following platforms:
  • Windows 32 and 64 bits with Java
  • Mac OS X 32 and 64 bits with Java
  • Linux x86, amd64, PPC, ARM, MIPS, SPARC, ALPHA, Itanium, SH4, etc with Java (from Oracle or OpenJDK)
  • Android 2.0+
